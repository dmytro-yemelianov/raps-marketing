## Documentation Version References

When updating marketing materials, use these version numbers:

### For Press Releases
- RAPS Version: "4.0+" (general) or "4.2.1" (specific)
- APS API: "v1/v2 APIs" or "current APS APIs"
- Year: "2026"

### For Cheat Sheets
- Always include specific RAPS version in header
- Reference specific API versions for commands
- Include compatibility notes for older versions

### For Technical Documentation
- Specify exact RAPS version and build
- List all supported APS API versions
- Include deprecation notices where applicable

## Version Update Process

1. **Quarterly Review** - Update version references
2. **Release Alignment** - Sync with RAPS release schedule  
3. **API Changes** - Monitor APS API updates from Autodesk
4. **Documentation Sync** - Ensure all materials reference correct versions
