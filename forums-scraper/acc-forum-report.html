ðŸ“Š Methodology

Data collection: We scraped the ACC Ideas forum (forums.autodesk.com/t5/acc-ideas/) in January 2026, collecting all publicly visible ideas with metadata including title, body, kudos, views, replies, status, and dates.

Analysis: Python (pandas) for data processing and categorization. Topic classification used keyword matching against 18 predefined category groups.

Limitations: Only public forum data (no internal Autodesk metrics). Keyword categorization may misclassify edge cases. "Gathering Support" status may include ideas Autodesk tracks internally.
