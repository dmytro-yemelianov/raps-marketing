## Quick Start

1. **Set Up Profile**: Use `profile/overview.md` for your Upwork bio
2. **Add Portfolio Items**: Upload projects from `portfolio/`
3. **Respond to Jobs**: Use templates from `proposals/templates/`
4. **Track Performance**: Document wins in `case-studies/`
