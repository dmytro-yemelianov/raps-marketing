# Profile Title Options

## Primary Title Recommendations

### Option 1: APS Focus (Recommended for Niche)
```
Autodesk Platform Services (APS/Forge) Expert | Rust Developer | AEC Automation
```
*Best for: Targeting APS-specific jobs*

### Option 2: Broader Appeal
```
Senior API Developer | Autodesk Forge/APS | CLI & Automation Specialist
```
*Best for: Casting wider net while maintaining specialty*

### Option 3: Enterprise Focus
```
AEC Technology Consultant | Autodesk Platform Services | Enterprise Integrations
```
*Best for: Attracting consulting and larger enterprise projects*

### Option 4: Technical Focus
```
Rust Developer | Autodesk APS Expert | DevOps & CI/CD Automation
```
*Best for: Technical leadership and architecture roles*

## Seasonal Adjustments

### Q1 (January-March)
Emphasize: New year projects, migrations, roadmap implementations

### Q2 (April-June)
Emphasize: Mid-year initiatives, scaling, optimization

### Q3 (July-September)
Emphasize: Pre-conference prep, proof of concepts

### Q4 (October-December)
Emphasize: Year-end projects, Autodesk University follow-up, budget spend

---

## Title Don'ts

❌ Don't include:
- Rates or pricing
- Vague terms like "Full Stack"
- Too many technologies
- Buzzwords without substance

❌ Avoid:
- "Freelancer" (implied on Upwork)
- "Available" or availability notes
- Personal branding slogans
